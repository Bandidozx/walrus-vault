📝 PROJECT NAME:
Walrus Vault

📋 DESCRIPTION (copy-paste ini):

Decentralized file storage platform built on Sui blockchain with Walrus integration and Tatum payment processing.

**Key Features:**
• Upload/download files with Walrus decentralized storage
• NFT-based access control via Sui Move smart contracts
• Tiered payment system (Free: 1GB, Basic: 10GB, Pro: 100GB, Enterprise: 1TB)
• Modern drag-and-drop UI with real-time file statistics
• Docker-ready deployment

**Tech Stack:**
• Backend: Express.js (Node.js) - 7 REST API endpoints
• Blockchain: Sui Move smart contract (deployed)
• Storage: Walrus decentralized storage with local fallback
• Payments: Tatum API integration
• Frontend: Responsive HTML5/CSS3/JavaScript
