📝 PROJECT NAME:
Walrus Vault

📋 DESCRIPTION:
Decentralized file storage platform built on Sui blockchain with Walrus integration and Tatum payment processing.

**Features:**
• Upload/download files with Walrus decentralized storage
• NFT-based access control via Sui Move smart contracts
• Tiered payment system (Free, Basic, Pro, Enterprise)
• Modern drag-and-drop UI
• Docker-ready deployment

**Tech Stack:**
• Backend: Express.js (Node.js)
• Blockchain: Sui Move
• Storage: Walrus
• Payments: Tatum API
• Frontend: HTML5/CSS3/JavaScript
