# Walrus Vault - Demo Script

## Video Duration: 3-5 minutes

### Scene 1: Introduction (30s)
**Visual**: Landing page with gradient background
**Narration**:
"Welcome to Walrus Vault - a decentralized file storage solution built on Walrus, Sui blockchain, and Tatum. This project was created for the Walrus x Sui x Tatum Hackathon 2026."

**Show**:
- Project title and tagline
- Hackathon logo
- Tech stack icons

### Scene 2: Problem Statement (30s)
**Visual**: Comparison chart
**Narration**:
"Traditional cloud storage is centralized, expensive, and lacks true ownership. Walrus Vault solves this by combining decentralized storage with blockchain-based access control."

**Show**:
- Centralized vs Decentralized comparison
- Key benefits: Security, Ownership, Transparency

### Scene 3: Architecture Overview (45s)
**Visual**: Architecture diagram
**Narration**:
"The architecture consists of three layers: Walrus for distributed file storage, Sui blockchain for access control via NFTs, and Tatum for payment processing."

**Show**:
- Frontend → Backend → Walrus
- Smart contract on Sui
- Payment flow with Tatum

### Scene 4: Live Demo - Upload (60s)
**Visual**: Screen recording
**Narration**:
"Let's see it in action. First, I'll upload a file by dragging it to the upload zone."

**Actions**:
1. Show wallet address and balance
2. Drag test file to upload zone
3. Show upload progress
4. Display success message with blob ID
5. Show file appears in list

### Scene 5: Live Demo - File Management (45s)
**Visual**: Screen recording
**Narration**:
"The dashboard shows all uploaded files with their size, storage type, and upload timestamp. I can download or delete files with one click."

**Actions**:
1. Show file list with stats
2. Click download button
3. Show file downloaded successfully
4. Demonstrate delete functionality

### Scene 6: Smart Contract (30s)
**Visual**: Code editor with vault.move
**Narration**:
"The Sui Move smart contract defines FileVault and AccessNFT structs. When you upload a file, an NFT is minted that grants access. You can transfer this NFT to share files securely."

**Show**:
- FileVault struct code
- AccessNFT struct code
- upload_file function

### Scene 7: Technical Highlights (30s)
**Visual**: Code snippets
**Narration**:
"Key features include: Walrus integration with local fallback, Sui wallet generation, drag-and-drop UI, and RESTful API with Express.js."

**Show**:
- API endpoints list
- Wallet integration code
- Upload handler code

### Scene 8: Future Roadmap (20s)
**Visual**: Roadmap timeline
**Narration**:
"Next steps include deploying the Move contract, integrating blockchain transactions, adding Tatum payments, and implementing file encryption."

**Show**:
- Deployment checklist
- Feature roadmap
- Timeline to June 6

### Scene 9: Closing (20s)
**Visual**: GitHub repo + submission info
**Narration**:
"Thank you for watching! Check out the code on GitHub and follow the project. Built for the Walrus x Sui x Tatum Hackathon 2026."

**Show**:
- GitHub URL: github.com/Bandidozx/walrus-vault
- Hackathon details
- Contact info

- [ ] Use clear audio (no background noise)
- [ ] Show cursor movements

- [ ] Include background music (optional)
- [ ] Export as MP4 (H.264)
- [ ] Upload to YouTube (unlisted)

## Tools Needed

- Screen recorder: OBS Studio / QuickTime
- Video editor: DaVinci Resolve / iMovie
- Audio: Built-in mic or external
- Diagrams: Excalidraw / draw.io

## Recording Tips

1. Close unnecessary tabs/apps
2. Clear browser cache
3. Test upload with small file first
4. Rehearse script 2-3 times
5. Record in quiet environment
6. Keep cursor movements smooth
7. Pause between scenes for editing
8. Record multiple takes if needed

## Post-Production

1. Trim dead space
2. Add transitions between scenes
3. Insert text overlays for key points
4. Adjust audio levels
5. Add intro/outro cards
6. Export at 1080p 30fps
7. Upload to YouTube
8. Add to submission document
